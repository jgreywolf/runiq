# HorizontalOrgChart

**Category:** glyphsets

A horizontal organization-chart glyphset that renders hierarchical org trees left-to-right. Provide a `hierarchy` data structure and optional `colors` and `nodeShape` parameters to control appearance.

## Example

```runiq
shape node1 as @horizontalOrgChart label:"Example HorizontalOrgChart"
```
