# OrgChart

**Category:** glyphsets

A vertical organization-chart glyphset that renders hierarchical trees top-to-bottom. Provide a `hierarchy` structure and optional `colors` and `nodeShape` to control rendering.

## Example

```runiq
shape node1 as @orgChart label:"Example OrgChart"
```
