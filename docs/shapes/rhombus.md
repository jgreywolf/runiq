# Rhombus

Diamond / decision shape used in flowcharts to represent branching logic. Accepts labels and edge annotations; supports `fillColor` and `strokeColor`.

**Category:** basic

## Example

```runiq
shape node1 as @rhombus label:"Example Rhombus"
```
