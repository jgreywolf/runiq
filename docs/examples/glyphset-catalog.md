# Glyphset Catalog

This catalog lists every built-in glyphset in Runiq with a thumbnail preview and sizing limits.

- Full reference: [Glyphsets Reference](/reference/glyphsets)
- Examples: [Glyphset Diagram Examples](/examples/glyphsets)

Previews are generated from the registry using `scripts/generate-glyphset-previews.mjs`. The source JSON files live in `docs/examples/glyphset-catalog-sources/` and the SVGs live in `docs/public/examples/glyphsets/`.
